About CareSync

Concise overview of the product, architecture, and key capabilities.

Multi-role: Users & Doctors Real-time: Socket.IO + WebRTC Secure by design

What CareSync Is

CareSync is a focused digital-health project that demonstrates patient–doctor workflows: account creation, scheduling, video consultations, messaging, prescriptions, and medication reminders. It also includes optional AI assistance for summarization and guidance.

Architecture

  • Python + Flask
  • Flask-SQLAlchemy (ORM)
  • Flask-SocketIO
  • WebRTC (media)
  • Flask-WTF (CSRF)
  • Flask-Migrate (Alembic)

Dev storage uses SQLite; production can switch to a managed database via environment config.

Data Handling

Passwords are stored using strong one-way hashing. Uploads are stored on the server and validated. Session and security headers are configured to reduce common risks. See the Privacy Policy for details.

AI Assistance

Integrates with Google’s Gemini for text and multimodal prompts. Interaction history may be stored to provide context and continuity.

Consultations

WebRTC for audio/video. Socket.IO handles signaling and room state. Optional watermarking and temporary storage for files shared in sessions.

Scheduling & Reminders

Appointments and medication reminders with email notifications. Background tasks run on a scheduler to send time-based messages.

Location Utilities

Map views for clinics and a blood-bank locator using distance calculations to surface nearby resources.

Project Status

CareSync is a course project intended for demonstration and evaluation. It is not a medical device and is not intended for real-world clinical use.
